# Instrucciones para correr el programa
- Descargar el proyecto (click en code, download zip)
- Extraer el proyecto
- Instalar las dependencias indicadas en "Archivos Adicionales/Requirements.txt"
- Abrir visual code
- selecionar file, luego open folder y luego  elegir el proyecto 
- Dar click en main.py
- (Si es necesario, darle a descargar la libreria)
- Dar click en run ▶️ (parte superior derecha)
